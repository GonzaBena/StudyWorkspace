# Specs

la primera pantalla sera una card redondeada con bordes de color retro, brillo de atras
la card tendra un indicador de drag and drop
la card sera tambien un boton
al pulsar la card se abrira un selector de archivos pdf o carpetas
se guardara una lista con los pdf seleccionados o dentro de la carpeta en cuestion
una vez cargado se abrira el primer archivo con un pdf previewer dentro de la pagina
arriba de todo habra una status bar
la statusbar tendra una barra de progreso que indicara cuanto falta para terminar el documento
la statusbar tendra una barra de progreso que indicara cuanto falta para terminar el la lista de archivos
las barras de progreso estaran una sobre otra y seran pequeñas para no ocupar mucho
a la derecha del grupo de barras habra un boton para cerrar la sesion y volver al inicio con el boton de archivos.
las ultimas sesiones se guardaras localmente por si se quiere seguir con la anterior
al iniciar o reanudar una sesion se verificara la consistencia de los archivos para no intentar leer un archivo eliminado
las status bar iran cambiando de color segun el progreso
cada vez que se complete un archivo o lista saldra un efecto de confetti de felicidades
la pagina tendra animaciones usando animejs
la pagina tendra modo oscuro y modo claro
se podra cambiar de modo oscuro a claro con un boton flotante abajo a la derecha
la pagina debe adaptarse a mobiles y escritorio
